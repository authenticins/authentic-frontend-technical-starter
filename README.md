# Authentic Frontend Technical (In-person)

Create a React/TS app that fetches a random word from the [API](https://random-word-api.herokuapp.com/home) and displays it to the user.

##### Minimum requirements:
- Use a single, functional component *(1 point)*
- Use React hooks (useState, useEffect) as necessary *(1 point)*
- Declare types/interfaces/enums as necessary *(1 point)*
- Fetch an initial word on mount *(1 point)*
- Provide a button for the user to get new words *(1 point)*

##### Bonus points:
- Allow user to specify word length *(1 point)*
- Display some sort of loading state as new words are fetched *(1 point)*
- Style the app using CSS-in-JS *(1-2 points based on design)*
